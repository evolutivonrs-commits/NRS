# **App Name**: Lexis Pulse

## Core Features:

- Smart Spaced-Repetition Engine: An AI-powered tool that utilizes a specialized algorithm to calculate optimal review intervals (SM-2 logic) for notebook entries based on user retention scores.
- Intelligent Study Dashboard: A centralized hub displaying retention rates, streaks, and dynamic counters for pending reviews using real-time Firestore synchronization.
- Cloud-Synced Notebooks: Organizational structure for subjects like Law or English with secure data persistence and Google Sign-in.
- 15-Minute Flash Session: A focused study mode that automatically selects high-priority cards for quick review bursts.
- Global Knowledge Search: Instant filtering of notes, tags, and media across all notebooks.
- Academic Data Export: Feature to generate comprehensive study reports and content logs in PDF and Excel formats.
- Visual Study Timeline: An interactive monthly calendar to track historical performance and plan future review milestones.

## Style Guidelines:

- Primary Color: Intellectual Indigo (#6366F1), evoking focus and cognitive depth. Used for interactive elements and brand identifiers.
- Background: Deep Navy (#0F172A), providing a premium, high-contrast environment suitable for prolonged study sessions without eye strain.
- Accent: Vivid Sky Blue (#0EA5E9), used strategically for critical CTA buttons like 'Start Study' and retention indicators.
- Headline Font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and academic look. Body Font: 'Inter' (sans-serif) for maximum legibility in longer notes and summaries.
- Sharp, thin-stroke linear icons with subtle duotone fills to represent various subjects and calendar statuses.
- A 'Notion-meets-Dashboard' layout using a clean sidebar for navigation and card-based containers for content grouping.
- Micro-interactions for button presses and smooth layout transitions (fade and slide) when switching between notebook view and review sessions.
